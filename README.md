# ContactApp

## Description

ContactApp is a simple Android application that allows users to add and save contact information. It features a clean user interface for inputting details such as name, phone number, email, address, and category.

## Features

*   **Add Contact:** A form to enter the following contact details:
    *   Full Name
    *   Phone Number
    *   Email Address
    *   Address (2 lines)
*   **Categorization:** Assign a category to each contact:
    *   Friend (Ami)
    *   Family (Famille)
    *   Work (Travail)
*   **Reset:** Clear all the fields in the form.
*   **Display:** Show the details of the added contact on the screen.

## Technologies Used

*   **Language:** Kotlin
*   **UI Framework:** Android SDK with XML layouts
*   **Layout:** ConstraintLayout for responsive screen design.

## How to Get Started

1.  Open the project in Android Studio.
2.  Build the project to install all the required dependencies.
3.  Run the app on an Android emulator or a physical device.



# tpdistance1 - Simple Calculator

## Description

`tpdistance1` is a basic Android calculator application that allows users to perform simple arithmetic operations. The app provides a user-friendly interface to input two numbers and calculate their sum or difference.

## Features

*   **Number Input:** Two input fields for entering numbers.
*   **Addition:** A button to add the two numbers together.
*   **Subtraction:** A button to subtract the second number from the first.
*   **Result Display:** The result of the calculation is displayed on the screen.
*   **Input Validation:** The app includes a check to ensure that both number fields are filled before performing a calculation.

## Technologies Used

*   **Language:** Kotlin
*   **UI Framework:** Android SDK with XML layouts
*   **Key Dependencies:**
    *   `androidx.appcompat`
    *   `com.google.android.material:material`
    *   `androidx.constraintlayout`

## How to Get Started

1.  Open the project in Android Studio.
2.  Build the project to ensure all dependencies are correctly installed.
3.  Run the application on an Android emulator or a physical device.



# Profil App

## Description

Profil is a simple Android application that allows users to view and edit their profile information. The app provides a straightforward interface for updating a user's name, email, and phone number.

## Features

*   **View Profile:** Displays the user's current name and email.
*   **Edit Profile:** Provides fields to edit the following details:
    *   Name (Nom)
    *   Email
    *   Phone Number (Téléphone)
*   **Save Profile:** A button to save the updated profile information.
*   **Input Validation:** The app checks to ensure that the name, email, and phone number fields are not empty before saving.

## Technologies Used

*   **Language:** Kotlin
*   **UI Framework:** Android SDK with XML layouts
*   **Layout:** ConstraintLayout for a responsive UI.

## How to Get Started

1.  Open the project in Android Studio.
2.  Build the project to download and install all the necessary dependencies.
3.  Run the app on an Android emulator or a physical device.
